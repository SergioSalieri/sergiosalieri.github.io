Задание №1

Это задание на тему "Работа с инструментами аналитика информационной безопасности".

Цель задания: познакомиться с инструментами разведки компонентов и командной строки в Astra Linux и Windows. Научиться настраивать безопасность сетевых устройств.

В результате выполнения задания я научился: проводить анализ уязвимостей своей операционной системы и сетевых устройств; находить какие компоненты для обеспечения безопасности операционной системы следует настроить.

Задание выполнено за 3 часа.

Задание №2

Это задание на тему «Белый хакинг: тестирование на проникновение».

Цель задания: Провести тестирование на проникновение в современных веб-технологиях, анализировать уязвимости.

В результате выполнения задания я научился: обнаруживать, эксплуатировать и тестировать веб-технологии на основе методологии OWASP.

Задание выполнено за 3 часа.

Задание №3

Это задание на тему «Настройка аудита, написание powershell скрипта для Active Directory, регулярные выражения для событий».

Цель задания: Сконфигурировать клиент-серверную модель для отправки логов с клиента на сервер. Написать скрипт для вывода информации о пользователях в виде таблицы. Написать регулярные выражения для событий.

В результате выполнения задания я научился: Правильно конфигурировать отправку логов на сервер, писать powershell скрипт для Active Directory, писать регулярные выражения.

Задание выполнено за 3 дня.
